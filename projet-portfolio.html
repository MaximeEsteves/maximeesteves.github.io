"Projets"

La suite du site web montre les projets en détails. Vous avez deux options pour avoir accès aux œuvres. La première est de cliquer sur le thème en question pour être redirigé vers tous les ouvrages de ce thème. La seconde option consiste à cliquer directement sur la miniature du projet que vous souhaitez regarder pour être directement dirigé vers l’œuvre en question.
